# Hearthscape GO

*ONLY use on Greenfoot 2.4.2 (Any newer, or older versions may will not work)

open project.greenfoot to launch game

## Controls
```
Mouse - Shoot
Spacebar - Mine resources
W - Move up
A - Move left
S - Move down
D - Move right
```
